## Design Context

### Users
研究人员、开发者、CV 从业者，有一定技术背景，在本地环境使用，目标是快速完成"标注→训练→推理"闭环。公开开源工具，用户水平参差，但核心受众是自己动手的技术用户。

### Brand Personality
工具性、克制、可信赖。三个词：**精准、沉稳、高效**。
界面不应抢镜，它只是达成目标的手段。情感目标：让用户感到"一切都在掌控中"。

### Aesthetic Direction
- 深度参考 Apple/iOS HIG：SF Pro 字体栈、8px 圆角体系、玻璃毛边侧栏/顶栏
- 深色优先，同时支持浅色（系统跟随 + 手动切换）
- 极简主义：无装饰性元素，依赖间距、半透明、细边线传达层次
- 单色主调（深灰系），红色仅用于破坏性操作
- 反例：不用霓虹发光、不用彩色渐变、不用卡片嵌卡片、不用圆角矩形 + 彩色单侧边框

### Design Principles
1. **工具优先**：每个像素都要为任务服务，装饰只在增强可用性时才出现
2. **层次靠密度**：用间距和透明度区分主次，而非颜色堆砌
3. **操作零摩擦**：常用路径不超过两步，弹窗/下拉只在必要时出现且立即消失
4. **一致性胜于创意**：新组件必须复用现有 CSS 变量和圆角体系，不引入新的视觉语言
5. **深浅模式同等公民**：每个组件都要在两种主题下测试
